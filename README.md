# MindGuard MVP (улучшенный)

PWA-прототип для осознанности, ежедневных отчётов, аналитики и жёсткого Focus/Blocker режима.

## Что улучшено
- Более полный дневной отчёт: здоровье, сознание, ЕГЭ, рефлексия + дополнительные поля (время сна, питание/КБЖУ, вода, витамины/лекарства, контроль/контрмера, и т.д.).
- Режимы отчёта `short/full`, шаблоны с кастомными вопросами и автосохранение черновика.
- Редактирование из History + фильтрация по шаблону и поиску.
- Расширенные практики осознанности:
  - box-breathing cycles,
  - urge surfing,
  - body scan,
  - медитация/дневник/благодарность.
- Аналитика 7/30 дней:
  - средние,
  - графики сна/энергии/стресса/настроения/веса/шагов/deep work/дыхания,
  - простые корреляции,
  - toggle для insights.
- Focus & Blocker:
  - in-app Focus Browser с blocklist/allowlist,
  - strict levels 1-4,
  - Unlock Flow: задержка + задание + подтверждение + partner code на L4,
  - Emergency Access (1 раз в N дней, логирование 24h ожидания),
  - PIN-защита настроек,
  - журнал попыток.
- Offline-first: localStorage + syncQueue при возврате сети.
- Подготовка V2: интерфейс `SystemBlockerAdapter` для системного модуля (расширение/VPN/Accessibility).

python3 -m http.server 4173
